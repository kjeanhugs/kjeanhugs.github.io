Publishing a Website on GitHub Pages

Step 1: Create a GitHub account

If you don't already have one, create a GitHub account by visiting https://github.com/ and clicking on "Sign up".

Step 2: Create a new repository

Once you've logged in to GitHub, create a new repository by clicking on the "+" icon in the top right corner and selecting "New repository". Give your repository a name in the format "username.github.io", where "username" is your GitHub username.

Step 3: Clone the repository to your local machine

Use the "Clone or download" button to copy the repository's URL, and then open a terminal and run the command "git clone repository_url", where "repository_url" is the URL you just copied.

Step 4: Add your website files to the repository

Copy the files for your website (HTML, CSS, JavaScript, images, etc.) into the repository's root directory. You can use your code editor or the command line to do this.

Step 5: Commit and push your changes

In the terminal, navigate to the repository's root directory and run the following commands:

  • "git add ."
  • "git commit -m 'Initial commit'"
  • "git push origin master"

This will upload your website files to GitHub.

Step 6: View your website

Your website should now be live at "https://username.github.io", where "username" is your GitHub username.
